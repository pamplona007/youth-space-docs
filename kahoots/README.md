## Como Usar

1. Acesse [kahoot.com](https://kahoot.com) e faça login
2. Clique em **Create** → **Kahoot**
3. Selecione as perguntas do arquivo markdown correspondente ao módulo
4. Cada Kahoot recomendado tem **10-12 perguntas** para 10-15 minutos de jogo

## Dicas de Uso

- **Início da aula**: Use 5-10 perguntas como warm-up (10 min)
- **Meio da aula**: Use para quebrar monotonia e revisar conteúdo (10 min)
- **Final da aula**: Use como fechamento e verificar aprendizado (10-15 min)
- **Aula rápida**: Se sobrar tempo no final, 5 perguntas rápidas para engajamento

## Formato das Perguntas

Cada pergunta está no formato:
```
**P1.** Pergunta?
- A) Opção
- B) Opção
- C) Opção
- D) Opção ✅
```

O emoji ✅ indica a resposta correta.

## Contribuindo

Para adicionar/atualizar perguntas:
1. Edite o arquivo `.md` do módulo correspondente
2. Mantenha o formato: `**P#.** pergunta + 4 opções + ✅ na correta`
3. Atualize o README se adicionar novos Kahoot(s)
